# 教程弹窗UI优化 Spec

## Why
用户反馈CodeLab页面的教程详情弹窗存在两个主要问题：
1. 弹窗位置偏上，被顶部导航栏遮挡，影响内容查看
2. 弹窗缺乏动画效果，打开/关闭和切换教程时体验不够流畅，需要添加过渡动画提升用户体验

## What Changes
- 调整教程弹窗的垂直位置，确保不被导航栏遮挡
- 为弹窗背景遮罩添加淡入淡出动画
- 为弹窗内容区域添加缩放+淡入的弹出动画
- 为上一步/下一步按钮切换添加滑动过渡动画
- 优化移动端响应式表现

## Impact
- Affected specs: 无依赖其他规范
- Affected code: `d:\web\Winters\frontend\src\views\lab\CodeLab.vue`

## ADDED Requirements

### Requirement: 弹窗位置调整
系统 SHALL 确保教程弹窗在垂直方向上居中显示，且不被页面顶部导航栏遮挡。

#### Scenario: 弹窗正确显示位置
- **WHEN** 用户点击学习任务卡片打开教程弹窗
- **THEN** 弹窗应在视口内垂直居中显示，顶部留有足够间距避免被导航栏遮挡

### Requirement: 弹窗打开/关闭动画
系统 SHALL 为教程弹窗提供流畅的进入和退出动画效果。

#### Scenario: 弹窗打开动画
- **WHEN** 用户点击学习任务卡片
- **THEN** 弹窗背景遮罩从透明渐变到半透明黑色（0.3s）
- **AND** 弹窗内容从缩放0.95 + 透明度0 渐变到原始大小 + 完全不透明（0.3s）

#### Scenario: 弹窗关闭动画
- **WHEN** 用户点击关闭按钮或点击遮罩层
- **THEN** 弹窗内容从原始大小 + 不透明度1 渐变到缩放0.95 + 透明度0（0.2s）
- **AND** 弹窗背景遮罩从半透明黑色渐变到透明（0.2s）

### Requirement: 教程切换动画
系统 SHALL 在用户点击上一步/下一步按钮时提供平滑的内容切换动画。

#### Scenario: 切换到上一个教程
- **WHEN** 用户点击"上一步"按钮
- **THEN** 当前教程内容向右滑出并淡出（0.25s）
- **AND** 新教程内容从左侧滑入并淡入（0.25s）

#### Scenario: 切换到下一个教程
- **WHEN** 用户点击"下一步"按钮
- **THEN** 当前教程内容向左滑出并淡出（0.25s）
- **AND** 新教程内容从右侧滑入并淡入（0.25s）

## MODIFIED Requirements

### Requirement: 响应式设计
修改现有的响应式样式规则，确保在小屏幕设备上弹窗动画同样流畅且不影响性能。
