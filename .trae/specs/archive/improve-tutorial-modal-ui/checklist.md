# Checklist

- [x] 弹窗在桌面端垂直居中显示，顶部不被导航栏遮挡
- [x] 弹窗在移动端正确显示，内容可滚动
- [x] 打开弹窗时背景遮罩从透明渐变到 rgba(0,0,0,0.8)
- [x] 打开弹窗时内容从 scale(0.95) + opacity(0) 过渡到正常状态
- [x] 关闭弹窗时内容从正常状态过渡到 scale(0.95) + opacity(0)
- [x] 关闭弹窗时背景遮罩渐变到透明
- [x] 点击"上一步"按钮时内容向右滑出，新内容从左滑入
- [x] 点击"下一步"按钮时内容向左滑出，新内容从右滑入
- [x] 切换动画时长约 0.25s，使用 ease-out 缓动
- [x] 弹窗动画时长约 0.3s 进入 / 0.2s 退出
- [x] 动画过程中无卡顿或闪烁现象
- [x] 快速连续点击不会导致动画错乱
- [x] 移动端动画表现与桌面端一致
