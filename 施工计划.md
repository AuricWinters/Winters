## 阶段 1：Easy 动效 Composables（纯 CSS / 简单事件）

每个做成独立文件，放 `src/composables/`。

### 1.1 `src/composables/useSpotlight.js`
```
<input>: 绑定的元素 ref
<effect>: @mousemove → el.style.setProperty('--mx','--my')
<CSS>: background: radial-gradient(circle at var(--mx) var(--my), rgba(var(--primary-rgb),0.12), transparent 50%)
```

### 1.2 `src/composables/useTilt.js`
```
<input>: 绑定的元素 ref
<effect>: @mousemove → perspective 800px + rotateX/Y (±15deg)
<CSS>: transition: transform 0.1s ease-out
```

### 1.3 `src/composables/useMagnet.js`
```
<input>: 绑定的元素 ref
<effect>: window @mousemove → 计算距元素中心距离 → translate
<CSS>: transition: transform 0.3s cubic-bezier(0.34,1.56,0.64,1)
```

### 1.4 `src/composables/useClickSpark.js`
```
<effect>: document @click → Canvas 2D 粒子爆发（12个/click）
<实现>: 全局挂一个 <canvas>，z-index 9999，pointer-events:none
```

### 1.5 `src/components/ShinyText.vue`
```
<props>: text, speed(默认4s)
<CSS>: @keyframes shine，background-clip: text
```

### 1.6 `src/components/GlitchText.vue`
```
<props>: text
<CSS>: ::before/::after 伪元素 + clip-path + @keyframes
```

### 1.7 `src/components/StarBorder.vue`
```
<props>: color(默认var(--primary)), speed(默认4s)
<CSS>: ::before 伪元素 + conic-gradient + rotate
```

### 1.8 `src/components/GlareHover.vue`
```
<slot>: 默认插槽
<CSS>: ::after 伪元素 + skewX + background-position 过渡
```

### 1.9 `src/components/GradientText.vue`
```
<props>: text, colors(默认primary→pink→blue→green)
<CSS>: background-clip: text + background-size 300% + @keyframes
```

### 1.10 阶段验证
- [ ] `npx vite build` 零错误
- [ ] `git commit -m "feat: Easy 动效 8 件套 — Spotlight/Tilt/Magnet/ClickSpark/ShinyText/GlitchText/StarBorder/GlareHover"`

## 阶段 2：Medium 组件（Canvas / IntersectionObserver / SVG）

### 2.1 `src/composables/useBlurReveal.js`
```
<effect>: IntersectionObserver → 逐字 filter:blur(8px)→0 + opacity 0→1
```

### 2.2 `src/composables/useCountUp.js`
```
<input>: target number, duration
<effect>: requestAnimationFrame + easeOutExpo
```

### 2.3 `src/composables/useBounceEntry.js`
```
<effect>: IntersectionObserver → scale(0→1) + cubic-bezier(0.34,1.56,0.64,1)
```

### 2.4 `src/composables/useBorderGlow.js`
```
<input>: 绑定的元素 ref
<effect>: @mousemove → 计算角度 → --angle 变量
<CSS>: conic-gradient mask + ::after 伪元素
```

### 2.5 `src/components/DecayCard.vue`
```
<effect>: SVG feTurbulence + feDisplacementMap，hover 时 animate scale 0→30→0
```

### 2.6 `src/components/DotField.vue`
```
<effect>: Canvas 2D 点阵 + mousemove 排斥力 + RAF
```

### 2.7 `src/components/MasonryGrid.vue`
```
<effect>: ResizeObserver → 计算每列高度 → absolute 定位
```

### 2.8 `src/components/BounceCards.vue`
```
<effect>: IntersectionObserver → 错时弹性入场 + hover 推开相邻
```

### 2.9 `src/components/AnimatedList.vue`
```
<effect>: IntersectionObserver → 列表项交错 fade + scale 入场
```

### 2.10 `src/components/CircularText.vue`
```
<effect>: CSS transform: rotate(calc(var(--i)*angle)) + 持续旋转
```
